Fundamentos de Programação em Python

Estruturas de Seleção

Permitem executar código com base em condições.

Aplicação: Escolher entre opções, como verificar se um usuário é maior de idade.

idade = 20
if idade >= 18:
    print("Maior de idade")
else:
    print("Menor de idade")

Estruturas de Repetição

Executam um bloco de código várias vezes.

Aplicação: Listar itens de um carrinho.

for i in range(5):
    print(f"Item {i+1}")

Vetores e Matrizes

Armazenam coleções de dados.

Aplicação: Gerenciar listas de produtos.

produtos = ["Camiseta", "Calça"]
matriz = [[1, 2], [3, 4]]
print(produtos[0])  # Camiseta

Funções e Procedimentos

Blocos de código reutilizáveis.

Aplicação: Calcular o total de um carrinho.

def calcular_total(precos):
    return sum(precos)

Tratamento de Exceção

Gerencia erros no programa.

Aplicação: Evitar falhas ao acessar um arquivo.

try:
    with open("arquivo.txt", "r") as f:
        print(f.read())
except FileNotFoundError:
    print("Arquivo não encontrado")
